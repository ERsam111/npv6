# JCG Supply Chain Analytics - Standalone Desktop Application

## Overview

This is a **complete offline standalone desktop application** for advanced supply chain analytics. It requires **no internet connection** and stores all data locally on your computer.

## Key Features

### 100% Offline Operation
- No internet connection required
- No cloud services or external APIs needed
- All data stored locally on your device
- Complete privacy - your data never leaves your computer

### Comprehensive Analytics Suite
1. **Demand Forecasting**
   - Multiple forecasting models (Moving Average, Exponential Smoothing, Linear Regression)
   - Scenario planning with manual adjustments
   - Price elasticity analysis
   - Historical data analysis

2. **Green Field Analysis (GFA)**
   - Distribution center location optimization
   - Customer demand visualization
   - Distance and cost analysis
   - Profitability analysis

3. **Inventory Optimization**
   - Stock level optimization
   - Simulation and scenario analysis
   - Multi-product inventory management

4. **Network Analysis**
   - Supply chain network optimization
   - Route optimization
   - Cost-to-serve analysis

## Quick Start

1. Install the application for your operating system
2. Launch "JCG Supply Chain Analytics"
3. Start with any module:
   - Upload your data (Excel/CSV)
   - Configure parameters
   - Run analysis
   - Export results

All your work is automatically saved locally.

## Data Privacy

- All data is stored in your local application data folder
- No data transmission to external servers
- Complete control over your sensitive business data
- Export your data anytime in standard formats

## Technical Details

- **Framework**: Electron + React + TypeScript
- **UI**: Tailwind CSS + shadcn/ui components
- **Storage**: Browser localStorage (embedded in Electron)
- **No Dependencies**: No external services, no API keys needed

## System Requirements

- **OS**: Windows 10+, macOS 10.13+, or Linux (Ubuntu 18.04+)
- **RAM**: 4GB minimum (8GB recommended)
- **Storage**: 500MB free space
- **Internet**: Not required (only for initial download/installation)
